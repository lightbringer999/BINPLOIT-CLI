# BINPLOIT-CLI
binploit-cli is a Linux-based binary analysis and exploitation assistant for educational, research, and CTF use. It automates stack and heap vulnerability analysis, offset discovery, and controlled local execution for user-owned binaries, helping users understand memory bugs, ELF internals, and libc behavior without blind exploitation.
